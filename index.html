O que é CLIENT?

Cliente-side( lado cliente) é um dispositivo que esta consumindo alguma informação na internet, este dispositivo se conecta na internet através de um computador que é chamado de servidor.

Qual a diferença?

O cliente consome as informações.

O servidor fornece as informação através da internet.

O que são arquivos estáticos?

São as informações buscadas pelo client no servidor que voltam em forma de arquivos (HTML, CSS, Javascript, Imagens entre outras formatações.

Aplicação Web

São aplicações web sites acessados pelo navegador como exemplo www.facebook.com, não são aplicações web quando são acessados por um aplicativo instalado em seu dispositivo.

O que são SERVERS?

O cliente faz uma requisição local LAN em HTTP e retorna em arquivos estáticos.

Tipos de servidor

PROXY: servido que é definido regras para acessar (bloqueia sites que não se enquadrem).

FIREWALL: sistema que impede entrada de computadores externos.

Email SERVER: armazena emails.

Database server: banco de dados.

Itranet: rede que se comunica com o servidor sem utilizar a internet, só acessível pela rede local.

LINGUAGEM SERVER-SIDE

Linguagens que são executadas no servidor.

LINGUAGEM CLIENT-SIDE

Linguagens que são executadas no cliente.
